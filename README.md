# 🌐 Suzany Estética - Website

Este projeto consiste no desenvolvimento de um site institucional para um negócio real de estética, com o objetivo de apresentar serviços, fortalecer a presença digital e facilitar o contato com clientes.

## ✨ Sobre o projeto

O site foi desenvolvido para uma  cliente , focando em:

- Estrutura limpa e semântica com HTML
- Estilização moderna com CSS (uso de variáveis no :root)
- Layout responsivo
- Foco em experiência do usuário (UX)
- Botão direto para contato via WhatsApp

## 💡 Objetivo

Criar um sit eficiente, elegante e funcional que ajude a profissional a:

- Apresentar seus serviços
- Transmitir confiança
- Aumentar o número de clientes

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## 📱 Funcionalidades

- Seção de serviços com descrição
- Área "Sobre"
- Contato direto via WhatsApp
- Layout adaptado para diferentes dispositivos

## 🚀 Status do projeto

Versão inicial finalizada (v1.0)  
Melhorias futuras podem incluir animações, otimizações e novas funcionalidades.
