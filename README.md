## 📌 Visão Geral do Projeto

Este projeto foi desenvolvido para a profissional **Suzany Estética**. O desafio era criar uma vitrine digital que não fosse apenas bonita, mas funcional, focada em converter visitantes casuais em clientes reais. 

Diferente de projetos acadêmicos, este site foi pensado para o mercado real, atendendo requisitos de **SEO, Performance e Experiência do Usuário (UX)**.

---

## 🎯 Desafios e Soluções

### 1. Profissionalismo e Credibilidade
**Problema:** A cliente precisava transmitir confiança antes mesmo do primeiro contato.
**Solução:** Design limpo (Clean Design), paleta de cores harmoniosa e tipografia elegante, reforçando a autoridade da marca no setor de beleza.

### 2. Conversão Direta (Agendamentos)
**Problema:** O fluxo de contato era fragmentado.
**Solução:** Implementação de CTAs (Call to Action) estratégicos e um botão flutuante de WhatsApp vinculado diretamente à API, facilitando o agendamento imediato.

### 3. Acesso Mobile (Responsividade)
**Problema:** Mais de 90% dos acessos de clientes de estética vêm de dispositivos móveis.
**Solução:** Desenvolvimento com foco em *Mobile-First*, garantindo uma navegação fluida em qualquer tamanho de tela.

## 🛠️ Pilares Técnicos

| Tecnologia | Finalidade |
| **HTML5 Semântico** | Melhorar o ranqueamento no Google (SEO) e acessibilidade. |
| **CSS3 Moderno** | Uso de variáveis (`:root`) para fácil manutenção e escalabilidade do layout. |
| **JavaScript Nativo** | Manipulações de DOM e interações leves para manter o site ultra veloz. |
| **Vercel** | Deploy contínuo e hospedagem de alta performance. |
